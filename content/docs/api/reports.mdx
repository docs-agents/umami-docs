通过 API 使用报告。

**端点**

```text
GET /api/reports

POST /api/reports/attribution

POST /api/reports/journey

POST /api/reports/revenue
POST /api/reports/utm
```

---

## 筛选

所有报告现在都可以使用请求体中的 filters 属性进行筛选。

**参数**

- `path`：（字符串 | 可选）URL 名称。
- `referrer`：（字符串 | 可选）引荐来源名称。
- `title`：（字符串 | 可选）页面标题名称。
- `query`：（字符串 | 可选）查询参数名称。
- `browser`：（字符串 | 可选）浏览器名称。
- `os`：（字符串 | 可选）操作系统名称。
- `device`：（字符串 | 可选）设备名称（例如 Mobile）。
- `country`：（字符串 | 可选）国家名称。
- `region`：（字符串 | 可选）地区/州/省名称。
- `city`：（字符串 | 可选）城市名称。
- `hostname`：（字符串 | 可选）主机名。
- `tag`：（字符串 | 可选）标签名称。
- `distinctId`：（字符串 | 可选）独立 ID 名称。
- `segment`：（uuid | 可选）细分的 UUID。
- `cohort`：（uuid | 可选）队列的 UUID。

**请求体**

```json
{
  "websiteId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
  "type": "attribution",
  "filters": { "os": "Mac OS", "device": "desktop" },
  "parameters": {
    "startDate": "2025-10-19T07:00:00.000Z",
    "endDate": "2025-10-26T06:59:59.999Z",
    "timezone": "America/Los_Angeles",
    "model": "first-click",
    "type": "path",
    "step": "/"
  }
}
```

## POST /api/reports/attribution

**参数**

- `websiteId`：（字符串）您的网站 id。

- `filters`：可以接受筛选参数。
- `startDate`：（数字）开始日期（以毫秒为单位）。
- `endDate`：（数字）结束日期（以毫秒为单位）。
- `model`：（字符串）归因模型（`firstClick` | `lastClick`）。
- `type`：（数组）转化类型（`path` | `event`）。

- `websiteId`：（字符串）您的网站 id。
- `type`：（字符串）报告类型（`attribution`| `breakdown`| `funnel`| `goal`| `journey`| `retention`| `revenue`| `utm`）。
- `filters`：可以接受筛选参数。
- `startDate`：（数字）开始日期（以毫秒为单位）。

**可用字段**

`path` | `title` | `query` | `referrer` | `browser` | `os` | `device` | `country` | `region` | `city` | `hostname`| `tag` | `event`

## POST /api/reports/funnel

了解用户的转化率和流失率。

**参数**

- `websiteId`：（字符串）您的网站 id。
- `type`：（字符串）报告类型（`attribution`| `breakdown`| `funnel`| `goal`| `journey`| `retention`| `revenue`| `utm`）。
- `filters`：可以接受筛选参数。
- `startDate`：（数字）开始日期（以毫秒为单位）。
- `endDate`：（数字）结束日期（以毫秒为单位）。
- `steps`：（数组 | 最少需要两个）事件类型（`path` | `event`）和转化步骤。
- `window`：（数字）漏斗步骤之间被视为转化的天数窗口。
