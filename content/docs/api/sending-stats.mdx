## POST /api/send

要注册一个 `event`，您需要向 `/api/send` 发送一个 `POST` 请求，
包含以下数据：

对于 **Umami Cloud**，向 `https://cloud.umami.is/api/send` 发送 POST 请求。

**参数**

`payload`

- `hostname`：（字符串）主机名。
- `screen`：（字符串）屏幕分辨率（例如 "1920x1080"）。
- `language`：（字符串）访问者语言（例如 "en-US"）。
- `url`：（字符串）页面 URL。
- `referrer`：（字符串）引荐来源 URL。
- `title`：（字符串）页面标题。
- `tag`：（字符串）附加标签描述。
- `id`：（字符串）会话标识符。
- `website`：（字符串）网站 ID。
- `name`：（字符串）事件名称。
- `data`：（对象 | 可选）事件的附加数据。

`type`：（字符串）`event` 是目前唯一可用的类型。

**示例 payload**

```json
{
  "payload": {
    "hostname": "your-hostname",
    "language": "en-US",
    "referrer": "",
    "screen": "1920x1080",
    "title": "dashboard",
    "url": "/",
    "website": "your-website-id",
    "name": "event-name",
    "data": {
      "foo": "bar"
    }
  },
  "type": "event"
}
```

请注意，对于 `/api/send` 请求，您不需要发送身份验证令牌。

此外，您需要发送正确的 `User-Agent` HTTP 标头，否则您的请求将不会被注册。

**示例响应**

```json
{
  "cache": "xxxxxxxxxxxxxxx",
  "sessionId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
  "visitId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
}
```

**以编程方式**

您可以使用浏览器 API 通过 JavaScript 以编程方式生成这些值。例如：

```js
const data = {
  payload: {
    hostname: window.location.hostname,
    language: navigator.language,
    referrer: document.referrer,
    screen: `${window.screen.width}x${window.screen.height}`,
    title: document.title,
    url: window.location.pathname,
    website: 'your-website-id',
    name: 'event-name',
  },
  type: 'event',
};
```
