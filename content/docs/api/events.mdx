事件和事件数据相关的操作。

**端点**

```text
GET /api/websites/:websiteId/events
GET /api/websites/:websiteId/event-data
GET /api/websites/:websiteId/event-data/:eventId
GET /api/websites/:websiteId/event-data/events
GET /api/websites/:websiteId/event-data/fields
GET /api/websites/:websiteId/event-data/properties
GET /api/websites/:websiteId/event-data/values
GET /api/websites/:websiteId/event-data/stats
```

---

## 筛选

所有标有 `filters` 的端点现在都可以使用以下参数进行筛选。

**参数**

- `path`：（字符串 | 可选）URL 名称。
- `referrer`：（字符串 | 可选）引荐来源名称。
- `title`：（字符串 | 可选）页面标题名称。
- `query`：（字符串 | 可选）查询参数名称。
- `browser`：（字符串 | 可选）浏览器名称。
- `os`：（字符串 | 可选）操作系统名称。
- `device`：（字符串 | 可选）设备名称（例如 Mobile）。
- `country`：（字符串 | 可选）国家名称。
- `region`：（字符串 | 可选）地区/州/省名称。
- `city`：（字符串 | 可选）城市名称。
- `hostname`：（字符串 | 可选）主机名。
- `tag`：（字符串 | 可选）标签名称。
- `distinctId`：（字符串 | 可选）独立 ID 名称。
- `segment`：（uuid | 可选）细分的 UUID。
- `cohort`：（uuid | 可选）队列的 UUID。

## GET /api/websites/:websiteId/events

获取给定时间范围内网站事件详情。

**参数**

- `startAt`：（数字）开始日期的时间戳（以毫秒为单位）。
- `endAt`：（数字）结束日期的时间戳（以毫秒为单位）。
- `search`：（字符串 | 可选）搜索文本。
- `page`：（可选数字，默认 1）确定页码。
- `pageSize`：（数字 | 可选，默认 20）确定返回的结果数量。

## GET /api/websites/:websiteId/event-data/events

获取事件数据名称、属性和计数

**参数**

- `startAt`：（数字）开始日期的时间戳（以毫秒为单位）。
- `endAt`：（数字）结束日期的时间戳（以毫秒为单位）。
- `event`：（字符串 | 可选）事件名称筛选。
- `filters`：可以接受筛选参数。

**示例响应**

```json
[
  {
    "eventName": "button-click",
    "propertyName": "id",
    "dataType": 1,
    "total": 4
  },
  {
    "eventName": "button-click",
    "propertyName": "name",
    "dataType": 1,
    "total": 4
  },
  {
    "eventName": "track-product",
    "propertyName": "price",
    "dataType": 2,
    "total": 2
  }
]
```

## GET /api/websites/:websiteId/event-data/properties

获取网站的事件名称和属性计数。

**参数**

- `startAt`：（数字）开始日期的时间戳（以毫秒为单位）。
- `endAt`：（数字）结束日期的时间戳（以毫秒为单位）。
- `filters`：可以接受筛选参数。

**示例响应**

```json
[
  {
    "eventName": "revenue-demo",
    "propertyName": "revenue",
    "total": 122
  },
  {
    "eventName": "revenue-demo",
    "propertyName": "currency",
    "total": 122
  }
]
```

## GET /api/websites/:websiteId/event-data/values

获取给定事件和属性的事件数据计数

**参数**

- `startAt`：（数字）开始日期的时间戳（以毫秒为单位）。
- `endAt`：（数字）结束日期的时间戳（以毫秒为单位）。
- `event`：（字符串）事件名称。
- `propertyName`：（字符串）属性名称。
- `filters`：可以接受筛选参数。

**示例响应**

```json
[
  {
    "value": "Male",
    "total": 28
  },
  {
    "value": "Female",
    "total": 26
  }
]
```

## GET /api/websites/:websiteId/event-data/stats

获取给定时间范围内聚合的网站事件、属性和记录。

**参数**

- `startAt`：（数字）开始日期的时间戳（以毫秒为单位）。
- `endAt`：（数字）结束日期的时间戳（以毫秒为单位）。
- `filters`：可以接受筛选参数。

**示例响应**

```json
[
  {
    "events": 16,
    "properties": 13,
    "records": 26
  }
]
```
