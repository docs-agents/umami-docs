---
title: 分享
---

分享页面管理相关的操作。

**端点**

```text
POST /api/share
GET /api/share/id/:shareId
POST /api/share/id/:shareId
DELETE /api/share/id/:shareId
GET api/websites/:websiteId/shares
POST api/websites/:websiteId/shares
```

---

## POST /api/share

创建分享页面。

**参数**

- `entityId`：（字符串）要添加的实体 ID（websiteId、pixelId、linkId 等）。
- `shareType`：（数字）（`website: 1` | `link:2` | `pixel: 3`）
- `name`：（字符串）分享页面名称。
- `slug`：（字符串）分享页面别名。
- `parameters`：（对象）分享页面参数。

**请求体**

```json
{
  "entityId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
  "shareType": 1,
  "name": "My Share Page",
  "slug": "abc123defg",
  "parameters": { "overview": true, "events": true}
}
```

**示例响应**

```json
{
  "id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
  "entityId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
  "name": "My Share Page",
  "shareType": 1,
  "slug": "abc123defg",
  "parameters": {
    "events": true,
    "overview": true
  },
  "createdAt": "2026-01-30T06:03:51.718Z",
  "updatedAt": "2026-01-30T06:03:51.718Z"
}
```

---

## GET /api/share/id/:shareId

通过 ID 获取分享页面。

**示例响应**

```json
{
  "id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
  "entityId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
  "name": "My Share Page",
  "shareType": 1,
  "slug": "abc123defg",
  "parameters": {
    "events": true,
    "overview": true
  },
  "createdAt": "2026-01-30T06:03:51.718Z",
  "updatedAt": "2026-01-30T06:06:32.197Z"
}
```

---

## POST /api/share/id/:shareId

更新分享页面。

**参数**

- `name`：（字符串）分享页面名称。
- `slug`：（字符串）分享页面别名。
- `parameters`：（对象）分享页面参数。

**请求体**

```json
{
  "name": "My Updated Share Page",
  "slug": "abc123defg",
  "parameters": { "overview": true, "events": true, "funnel": true, "utm": true}
}
```

---

## DELETE /api/share/id/:shareId

删除分享页面。

**示例响应**

```json
{
  "ok": true
}
```

---
