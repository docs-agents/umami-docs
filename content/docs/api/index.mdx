---
title: 概览
---

Umami 允许您通过调用应用程序的 API 端点直接获取数据。
您通过应用程序执行的任何操作都可以在 API 中使用。所有数据以 JSON 格式返回。

### 自托管

自托管时，API 端点位于 `http://<your-umami-instance>/api`。在调用之前，
您首先需要[进行身份验证](/docs/api/authentication)。

### Umami Cloud

如果使用 Umami Cloud，您将使用 [API 密钥](/docs/cloud/api-key) 调用 `https://api.umami.is`。
