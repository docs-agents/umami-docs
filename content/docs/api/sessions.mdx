title: 会话
---

会话和会话数据相关的操作。

**端点**

```text
GET /api/websites/:websiteId/sessions
GET /api/websites/:websiteId/sessions/stats
GET /api/websites/:websiteId/sessions/weekly
GET /api/websites/:websiteId/sessions/:sessionId
GET /api/websites/:websiteId/sessions/:sessionId/activity
GET /api/websites/:websiteId/sessions/:sessionId/properties
GET /api/websites/:websiteId/session-data/properties
GET /api/websites/:websiteId/session-data/values
```

---

## 筛选

所有标有 `filters` 的端点现在都可以使用以下参数进行筛选。

**参数**

- `path`：（字符串 | 可选）URL 名称。
- `referrer`：（字符串 | 可选）引荐来源名称。
- `title`：（字符串 | 可选）页面标题名称。
- `query`：（字符串 | 可选）查询参数名称。
- `browser`：（字符串 | 可选）浏览器名称。
- `os`：（字符串 | 可选）操作系统名称。
- `device`：（字符串 | 可选）设备名称（例如 Mobile）。
- `country`：（字符串 | 可选）国家名称。
- `region`：（字符串 | 可选）地区/州/省名称。
- `city`：（字符串 | 可选）城市名称。
- `hostname`：（字符串 | 可选）主机名。
- `tag`：（字符串 | 可选）标签名称。
- `distinctId`：（字符串 | 可选）独立 ID 名称。
- `segment`：（uuid | 可选）细分的 UUID。
- `cohort`：（uuid | 可选）队列的 UUID。

---

## GET /api/websites/:websiteId/sessions

获取给定时间范围内网站会话详情。

**参数**

- `startAt`：（数字）开始日期的时间戳（以毫秒为单位）。
- `endAt`：（数字）结束日期的时间戳（以毫秒为单位）。
- `search`：（字符串 | 可选）搜索文本。
- `page`：（数字 | 可选，默认 1）确定页码。
- `pageSize`：（数字 | 可选，默认 20）确定返回的结果数量。
- `filters`：可以接受筛选参数。

**示例响应**

```json
{
  "data": [
    {
      "id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
      "websiteId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
      "hostname": "umami.is",
      "browser": "chrome",
      "os": "Mac OS",
      "device": "desktop",
      "screen": "1800x1169",
      "language": "en-US",
      "country": "SE",
      "region": "SE-AB",
      "city": "Stockholm",
      "firstAt": "2025-10-21T13:35:51Z",
      "lastAt": "2025-10-21T15:00:09Z",
      "visits": 2,
      "views": 18,
      "createdAt": "2025-10-21T15:00:09Z"
    }
  ],
  "count": 923,
  "page": 1,
  "pageSize": 20
}
```

## GET /api/websites/:websiteId/sessions/stats

获取汇总的网站会话统计。

**参数**

- `startAt`：（数字）开始日期的时间戳（以毫秒为单位）。
- `endAt`：（数字）结束日期的时间戳（以毫秒为单位）。
- `filters`：可以接受筛选参数。

**示例响应**

```json
{
  "pageviews": { "value": 2924 },
  "visitors": { "value": 905 },
  "visits": { "value": 1050 },
  "countries": { "value": 84 },
  "events": { "value": 517 }
}
```

- `pageviews`：页面点击
- `visitors`：唯一访问者数量
- `visits`：唯一访问次数
- `bounces`：仅访问单个页面的访问者数量
- `totaltime`：在网站上花费的时间

---

## GET /api/websites/:websiteId/sessions/weekly

获取按工作日小时收集的会话计数。

**参数**

- `startAt`：（数字）开始日期的时间戳（以毫秒为单位）。
- `endAt`：（数字）结束日期的时间戳（以毫秒为单位）。
- `timezone`：时区（例如 America/Los_Angeles）。
- `filters`：可以接受筛选参数。
