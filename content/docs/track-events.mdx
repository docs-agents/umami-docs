---
title: 追踪事件
---

除了页面浏览，Umami 还可以追踪您网站上发生的事件。有两种方式在 Umami 中记录事件，使用 **data 属性**或使用 **JavaScript**。

## 限制

- 事件名称限制为 50 个字符。
- 没有事件名称无法发送事件数据。

## 使用数据属性

要启用事件，只需为您想要追踪的元素添加特殊的数据属性。

例如，您可能有一个如下代码的按钮：

```html
<button id="signup-button">Sign up</button>
```

使用以下格式添加数据属性：

```html
data-umami-event="{event-name}"
```

所以您的按钮元素现在应该是这样的：

```html
<button id="signup-button" data-umami-event="Signup button">Sign up</button>
```

当用户点击此按钮时，Umami 将记录一个名为 `Signup button` 的事件。

您可以选择使用 **data-umami-event-*** 注解传递 **event_data**。

```text
data-umami-event="Signup button"
data-umami-event-email="bob@aol.com"
data-umami-event-id="123"
```

附加属性将导致 `{ email: 'bob@aol.com', id: '123' }` 与 `Signup button` 名称一起记录。

### 注意事项

- 使用此方法时，所有事件数据将保存为字符串。如果您想将事件数据保存为数字、日期、布尔值等，请使用下面的 JavaScript 方法。
- 元素内的其他事件监听器不会被触发。

## 使用 JavaScript

您也可以使用 `window.umami` 对象手动记录事件。要完成与上述 data-* 方法相同的事情，您可以：

```js
const button = document.getElementById('signup-button');

button.onclick = () => umami.track('Signup button');
```

在这种情况下，Umami 将记录一个名为 `Signup button` 的事件。

如果您想记录动态数据，请参阅[追踪器函数](/docs/tracker-functions)。
