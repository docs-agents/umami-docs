---
title: 在 Koyeb 上运行
---

[Koyeb](https://www.koyeb.com) 是一个面向开发者的无服务器平台，可在全球范围内部署应用。该平台让您可以无缝运行 Docker 容器、Web 应用和 API，具有基于 Git 的部署、原生自动扩展、免费 SSL、全球边缘网络以及内置服务网格和服务发现功能。

## 设置

在本指南中，您需要有一个正在运行并接受远程连接的 PostgreSQL 数据库服务器。如果您还没有数据库，您可以查看[托管](/docs/guides/hosting)下的**托管数据库**部分。

> 您需要通过遵循[安装](/docs/install)文档的**创建数据库表**部分来设置数据库表。

为了部署 Umami，我们使用 [Koyeb CLI](https://github.com/koyeb/koyeb-cli)，您也可以使用 [Koyeb 控制面板](https://app.koyeb.com) 执行设置。

### 创建 Koyeb 密钥

创建一个 Koyeb 密钥以安全存储您的 `DATABASE_URL`。在终端中执行以下命令：

```shell
$ koyeb secret create umami-database-url
Enter your secret: postgres://<user>:<password>@<host>:<port>/<db-name>?sslmode=require

$ openssl rand -hex 32 | koyeb secret create umami-hash-salt --value-from-stdin
```

### 部署 Umami

创建密钥后，您可以部署 Umami。在终端中运行以下命令以创建新的 Koyeb 应用并部署 Umami 服务：

```shell
koyeb app init umami --docker docker.umami.is/umami-software/umami:postgresql-latest --ports 3000:http --routes /:3000 --env DATABASE_URL=@ umami-database-url
```

您的 Umami 服务正在部署中。要检索 Umami URL，请运行：

```shell
$ koyeb app get umami
ID                                  	NAME     	DOMAINS                         	UPDATED AT
30de8301-05b1-4131-a842-28e608900000	umami   	umami-<YOUR_KOYEB_ORG>.koyeb.app	2021-07-06 11:58:01.143967 +0000 UTC
```

打开 URL 访问 Umami 并按照**入门**指南从[登录](/docs/login)步骤开始。
