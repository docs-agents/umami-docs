---
title: 在 Qovery 上运行
---

[Qovery](https://www.qovery.com) 是一个完全托管的云平台，运行在您的 AWS、Digital Ocean 和 Scaleway 账户上，您可以在其中托管静态站点、后端 API、数据库、cron 作业以及您所有其他应用，全部在一个地方。

Qovery 为个人开发者提供**免费托管**，包括以下滚动块：

- 来自 GitHub 和 GitLab 的持续、自动构建和部署。
- 通过 [Let's Encrypt](https://letsencrypt.org) 自动获取 SSL 证书。
- 免费托管 PostgreSQL。
- 免费 SSD 存储。
- 无限协作者。
- 无限[自定义域名](https://hub.qovery.com/docs/using-qovery/configuration/application/#domains)。

## 设置

### 1. 创建 Qovery 账户

如果您还没有账户，请访问 [Qovery 仪表盘](https://start.qovery.com) 创建一个账户。

### 2. 创建一个项目

- 点击**创建项目**并为您的项目命名。
- 点击**下一步**。

![Create a project](https://hub.qovery.com/img/heroku/heroku-2.png)

### 3. 创建一个新环境

- 点击**创建环境**并命名（例如 staging、production）。

![Create a new environment](https://hub.qovery.com/img/heroku/heroku-3.png)

### 4. 添加您的 Umami 应用

- 点击**创建应用程序**，命名并选择您的 GitHub 或 GitLab 仓库，其中包含您的 Umami 应用。
- 定义主分支名称和根应用程序路径。
- 点击**创建**。

![Add your application](https://hub.qovery.com/img/rust/rust.png)

创建应用程序后：

- 导航到您的应用程序**设置**。
- 选择**端口**。
- 添加您的 Umami 应用程序使用的端口。

### 5. 部署数据库

按照[本指南](https://hub.qovery.com/guides/getting-started/create-a-database)创建和部署新的 PostgreSQL 数据库。

### 6. 添加存储

要添加存储，请进入您的应用程序**设置**：

![Add storage](https://hub.qovery.com/img/add-storage.png)

### 7. 设置您的 Umami 配置

要使用 Qivery 提供的 PostgreSQL，您可以使用内置的密钥和环境变量。您可以在我们的[配置部分](https://hub.qovery.com/docs/using-qovery/configuration/environment-variable/)中阅读更多关于环境变量和密钥的信息。

### 8. 在 Qovery 上部署应用

现在您需要做的就是在应用程序中导航并点击**部署**。

![Deploy the app](https://hub.qovery.com/img/heroku/heroku-1.png)

就是这样。观看状态并等待应用部署完成。

要在浏览器中打开应用程序，请在应用程序概览中点击**操作**和**打开**。
