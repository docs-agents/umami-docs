---
title: 托管
---

您可以通过多种方式启动和运行 Umami。
您只需要一个数据库 [PostgreSQL](https://www.postgresql.org/) 和可以运行 [Node.js](https://nodejs.org/)（18.18 或更高版本）的服务器。

## 单服务器

在此设置中，数据库和 Web 服务器运行在同一台机器上。您可以选择将 Umami 放在
[Nginx](https://www.nginx.com/) 或 [Apache](https://httpd.apache.org/) 等专用 Web 服务器后面，并将请求代理到 Umami。

您可以查看[在 DigitalOcean 上运行](/docs/guides/running-on-digitalocean)指南来了解如何设置服务器。

## 单独的数据库和 Web 服务器

在此设置中，数据库运行在自己的机器上，允许远程连接。您的 Web 服务器运行在不同的
机器上，安装了 Umami 并远程连接到数据库以存储和检索数据。

## 应用托管

在此设置中，您将让应用托管提供商为您托管 Umami 应用。您仍然需要在其他地方运行数据库，
以便应用可以连接。以下是一些可以托管 Umami 的应用托管提供商。Vercel、Netlify、Render 和 Northflank 有免费层可用。

- [Vercel](https://vercel.com/)
- [Netlify](https://www.netlify.com/)
- [Render](https://render.com/)
- [Northflank](https://northflank.com/)
- [Railway](https://railway.app/)

对于部署，构建命令是 `npm run build`，发布目录是 `.next`。
