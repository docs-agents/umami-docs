---
title: 在 Netlify 上运行
---

[Netlify](https://netlify.com/) 提供免费的托管服务，非常适合 GitHub 组织。

在此设置中，您应该已经有一个正在运行并接受远程连接的数据库服务器。如果您还没有数据库，
您可以按照[在 DigitalOcean 上运行](/docs/running-on-digitalocean)指南来启动和运行数据库。
您也可以查看[托管](/docs/guides/hosting)下的**托管数据库**部分。

## 设置

[![Deploy with netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/umami-software/umami)

_使用上面的按钮自动执行步骤 1-5_

1. 将 [https://github.com/umami-software/umami](https://github.com/umami-software/umami) 项目分叉到您的 GitHub 账户。
2. 在 [Netlify](https://netlify.com/) 上创建一个账户。
3. 从仪表盘页面点击**导入项目**，然后指定您的项目在 GitHub 上的分叉 URL。
4. 将构建命令更改为 **yarn run build**。
5. 添加必需的环境变量 `DATABASE_URL`。这些值在[安装](/docs/install)的**配置 Umami**步骤中定义。
6. 部署并访问您的应用程序。
7. 按照**入门**指南从[登录](/docs/login)步骤开始，并务必更改默认密码。
