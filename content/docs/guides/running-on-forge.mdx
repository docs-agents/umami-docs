---
title: 在 Forge 上运行
---

[Forge](https://forge.laravel.com/) 是一个付费的服务器管理服务，专为在 DigitalOcean、Linode、Vultr、Amazon、Hetzner 等服务器提供商上托管 Laravel 应用而设计。

如果您有 Forge 账户并连接了服务器存储桶，您可以轻松设置 Umami 安装。

## 先决条件

1. 配置您希望 Umami 可用的（子）域名。通常，您可以通过使用指向存储桶 IP 地址的 A 记录配置 DNS 来实现。

## 设置

1. 在 Forge 中选择您想要的服务器。
2. 进入**站点** > **新建站点**
   - 输入域名
   - 项目类型：静态 HTML
   - 勾选**创建数据库**，输入数据库名称
   - 点击**添加**
3. 要安装仓库，将 [https://github.com/umami-software/umami](https://github.com/umami-software/umami) 项目分叉到您的 GitHub 账户或从官方仓库安装。
   - 输入仓库路径：`umami-software/umami`
   - 取消勾选**安装 Composer 依赖**
   - 点击**安装仓库**
4. 更新部署脚本并在 `git pull origin $FORGE_SITE_BRANCH` 命令之后添加以下内容。

   ```shell
   yarn install
   yarn build
   ```

   删除以下行，因为我们没有 artisan 或使用 PHP：

   ```shell
   ( flock -w 10 9 || exit 1
   echo 'Restarting FPM...'; sudo -S service $FORGE_PHP_FPM reload ) 9>/tmp/fpmlock

   if [ -f artisan ]; then
       $FORGE_PHP artisan migrate --force
   fi
   ```

5. 在 Forge 中，进入您的服务器页面，点击**数据库**并为刚刚创建的数据库添加用户。
6. 创建数据库连接字符串并将其添加到您的 `.env` 文件：

   ```dotenv
   DATABASE_URL=mysql://user:password@localhost:3306/dbname
   ```

   连接 URL 格式如下：

   ```dotenv
   DATABASE_URL=postgresql://username:mypassword@localhost:5432/mydb
   DATABASE_URL=mysql://username:mypassword@localhost:3306/mydb
   ```

7. 通过 SSH 执行初始项目设置：

   ```shell
   yarn install
   yarn update-db
   yarn build
   ```

8. 进入您的 Umami 站点并在左侧菜单中点击**SSL**来添加 SSL 证书。
9. 点击**编辑文件** > **编辑 Nginx 配置**并覆盖以下块：

   ```nginx
   location / {
     proxy_pass http://localhost:3000;
     proxy_http_version 1.1;
     proxy_set_header Upgrade $http_upgrade;
     proxy_set_header Connection 'upgrade';
     proxy_set_header Host $host;
     proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
     proxy_cache_bypass $http_upgrade;
   }
   ```

   此配置将把流量路由到 Umami 运行所在的 node 服务器。

10. 在 Forge 中，进入您的服务器页面，点击**守护进程**并添加**新建守护进程**以保持 Umami Node 服务器运行。
    - 命令：`yarn start`
    - 目录：`/home/forge/umami.yourdomain.tld`
    - 点击**创建**

您现在应该能够访问您的 Umami 项目了。
