---
title: 在 PlanetScale 上运行
---

[PlanetScale](https://planetscale.com/) 是一个由 [Vitess](https://vitess.io/) 驱动的与 MySQL 兼容的无服务器数据库平台。

## 设置

1. 按照[安装](https://umami.is/docs/install)文档直到"配置 umami"部分。
2. 在 [PlanetScale](https://app.planetscale.com/) 上创建一个名为 `umami-db` 的数据库，并从"连接" > "使用 Prisma 连接" > ".env" 选项卡复制 `DATABASE_URL` 字符串。
3. 将 `DATABASE_URL` 添加到您的 `.env` 文件：

   ```dotenv
   DATABASE_URL=mysql://username:password@host/umami-db?sslaccept=strict
   ```

4. 运行 `yarn run build-db && yarn run update-db`
5. 访问 `umami-db` 仪表盘，确保表已创建，然后如果您愿意可以将其提升到生产环境。
6. 您现在应该能够构建并启动 Umami（先运行 `yarn build`，然后运行 `yarn start`）。
7. 按照**入门**指南从[登录](/docs/login)步骤开始，并务必更改默认密码。

### 故障排除

如果您遇到类似以下示例的错误：

```text
PrismaClientInitializationError: Can't reach database server at `host.aws-region.psdb.cloud`:`3306`
```

您可以尝试在您的 `DATABASE_URL` 上添加连接超时查询参数：

```dotenv
DATABASE_URL=mysql://username:password@host/umami-db?sslaccept=strict&connect_timeout=300
```
