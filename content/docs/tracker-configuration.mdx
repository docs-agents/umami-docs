---
title: 追踪器配置
---

Umami 追踪器提供了多个属性，允许您配置其行为。

### data-host-url

默认情况下，Umami 会将数据发送到脚本所在的位置。您可以覆盖此设置，将数据发送到另一个位置。

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-host-url="http://stats.mywebsite.com"
></script>
```

### data-auto-track

默认情况下，Umami 会自动追踪所有页面浏览和事件。您可以禁用此行为，并使用[追踪器函数](/docs/tracker-functions)自己追踪事件。

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-auto-track="false"
></script>
```

### data-domains

如果您希望追踪器仅在特定域名上运行，可以将它们添加到追踪脚本中。这是逗号分隔的域名列表。每个值与 `window.location.hostname` 匹配，因此您应该仔细检查您的网站是否使用 `www`。<br/>
如果您在预发布/开发环境中工作，这会有所帮助。<br/>

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-domains="mywebsite.com,www.mywebsite.com"
></script>
```

### data-tag

如果您希望追踪器在特定标签下收集事件。可以在仪表盘上按特定标签筛选事件。

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-tag="umami-eu"
></script>
```

### data-exclude-search

如果您不想从 URL 收集搜索参数。

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-exclude-search="true"
></script>
```

### data-exclude-hash

如果您不想收集 URL 中的哈希值。

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-exclude-hash="true"
></script>
```

### data-do-not-track

尊重用户的[请勿追踪](https://developer.mozilla.org/en-US/docs/Web/API/Navigator/doNotTrack)浏览器设置。

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-do-not-track="true"
></script>
```

### data-before-send

允许您指定一个在发送数据之前调用的函数。
然后您可以检查和修改 payload 或完全取消发送。
该函数将接受两个参数，type 和 payload。
要继续发送，请返回 payload 对象。
要取消发送，请返回一个假值。

```javascript
function beforeSendHandler(type, payload) {
  if (checkPayload(payload)) {
    return payload;
  }
  return false;
}
```

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-before-send="beforeSendHandler"
></script>
```
