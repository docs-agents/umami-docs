title: 订阅
---

Umami 使用 **Stripe** 作为支付处理器。注册计划时，您将被发送到 Stripe 结账门户。
所有订阅都附带 14 天免费试用。您可以在该时间范围内免费取消。

每个 Umami Cloud 计划的详细信息可以在[定价](https://umami.is/pricing)页面找到。

## 升级到 Pro

登录后，点击侧边导航下拉菜单，然后点击**设置**。

![image](/images/cloud/navbar-settings.png)

进入**计费**并点击**升级到 Pro**按钮。

![image](/images/cloud/billing-plans.png)

如果您对 **Enterprise** 计划感兴趣，请点击**联系我们**按钮或发送电子邮件至 sales@umami.is

您将被发送到 Stripe 结账门户以完成注册。进入**计费**查看您的最新计费详情。

![image](/images/cloud/billing-details.png)

## 管理计划

进入**计费**并点击**管理计划**按钮。您将被发送到 Stripe 计费门户，在那里您可以执行以下操作。

- 取消您的 **Pro** 计划
- 更新支付方式
- 更新计费信息
- 查看发票历史
- 下载发票和收据

![image](/images/cloud/billing-manage-plan.png)
