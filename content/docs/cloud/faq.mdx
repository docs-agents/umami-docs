# Umami Cloud 常见问题

### Umami Cloud 有免费层吗？

是的！Umami Cloud 的 Hobby 计划完全免费。非常适合个人项目和低流量网站。

### 免费试用如何运作？

Umami Cloud 的免费试用将从购买日期起持续 14 天。试用让您完全访问该计划中的所有功能。
试用结束时，您将按常规金额付费。您可以在试用期内随时取消试用，不收取任何费用。

### 我将如何被收费？

计费按月或按年进行。
如果您通过试用升级到付费计划，您的第一张发票将在试用期结束后到达。

### 如何衡量使用量？

使用量通过计算对网站的点击以及存储的任何自定义事件或自定义事件数据来衡量。每次网站点击计为一个事件。如果您保存事件数据，每个存储的数据属性计为一个事件。

### 您的服务器位于哪里？

Umami Cloud 服务器位于美国和欧盟，遵守 GDPR 和 CCPA 法规。

### 我可以导出我的数据吗？

是的，您可以从 Umami Cloud 导出所有数据。
