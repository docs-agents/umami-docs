Umami Cloud 是由 Umami 创建者运行的 Umami 应用程序的托管版本。

## 优势

- **完全托管**：获取所有最新更新，无需自行托管和管理您自己的服务器。
- **可扩展**：Umami 提供无缝可扩展性，允许您处理日益增长的数据量和用户需求，同时不影响性能。
- **成本效益高**：通过我们简单的按使用量计费模式，只为您使用的内容付费。
- **合规**：符合 GDPR 和 CCPA 法规。

## 注册

您可以在 [https://cloud.umami.is/signup](https://cloud.umami.is/signup?ref=umami-cloud-docs) 免费注册。

每个 Umami Cloud 计划的详细信息可以在[定价](https://umami.is/pricing)页面找到。
