---
title: 导出数据
---

Umami 允许您将数据从 Umami Cloud 导出为 CSV 格式（通过 gzip 压缩）。导出一次处理一个网站。
如果您有大量网站或大量数据要导出，请联系 [support@umami.is](mailto:support@umami.is)。

## 从 Umami Cloud 导出数据

登录后，点击侧边导航下拉菜单，然后点击**设置**。

![image](/images/cloud/navbar-settings.png)

进入**数据**并点击**导出**按钮。

![image](/images/cloud/export-data.png)

填写表单详情并点击**导出**按钮。

![image](/images/cloud/export-form.png)

导入完成后，您将收到一封包含导出文件下载链接的电子邮件。

![image](/images/cloud/export-email.png)
