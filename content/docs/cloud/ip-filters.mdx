---
title: IP 筛选器
---

Umami 允许阻止特定的 IP 地址或范围，因此您可以筛选传入流量并防止不需要或恶意连接。IP 筛选器从 **Pro** 计划开始可用。

## 添加 IP 筛选器

登录 Umami 并点击侧边导航下拉菜单，然后点击**设置**。

![image](/images/cloud/navbar-settings.png)

进入**IP 筛选器**并点击**添加 IP 筛选器**按钮。

![image](/images/cloud/ip-filter-add.png)

填写表单详情并点击**保存**按钮。

## 编辑或删除 IP 筛选器

进入**IP 筛选器**并点击您 IP 筛选器旁边的**编辑**按钮或**删除**按钮。

![image](/images/cloud/ip-filter-table.png)
