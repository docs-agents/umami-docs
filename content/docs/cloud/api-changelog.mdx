---
title: Umami API
---

此页面记录了对 **api.umami.is** 所做的更改。

## 2026-03-02

**事件**

- `GET /api/websites/:websiteId/event-data`：**新增**

## 2026-02-04

**分享**
- 在[此处](/docs/api/share)添加了 `share` API 文档。

## 2025-11-13

**实时**

- `GET /api/realtime/:websiteId`：移除 `timezone` 参数。响应将始终以 `UTC` 跨所有数据集返回。

## 2025-10-07

**筛选**

- 将筛选器 `url` 更改为 `path`。
- 将筛选器 `host` 更改为 `hostname`

**网站**

- `GET /websites/[id]/metrics`：`type` 参数 `url` 已更改为 `path`。
- `GET /websites/[id]/stats`：响应数据结构已更改。

**事件**

- `GET /event-data/values`：`eventName` 查询参数已更改为 `event`。

**报告**

- `POST /reports`：`insights` 报告已更改为 `breakdown`。

## 2024-09-09

**网站**

- `GET /websites/[id]/pageviews`：将 `region` 参数修复为可选。
- `GET /websites/[id]/events`：将结果集更改为给定时间范围内的网站事件详情。
- `GET /websites/[id]/events/series`：**新增**事件指标端点（之前的 `GET /websites/[id]/events`）。

**事件数据**

- `GET /event-data/events`：**移动**到 `/websites/[id]/event-data/events`。
- `GET /event-data/fields`：**移动**到 `/websites/[id]/event-data/fields`。
- `GET /event-data/stats`：**移动**到 `/websites/[id]/event-data/stats`。
- `GET /websites/[id]/event-data/values`：**新增**

**会话**

- `GET /websites/:websiteId/sessions`：**新增**
- `GET /websites/:websiteId/sessions/stats`：**新增**
- `GET /websites/:websiteId/sessions/:sessionId`：**新增**
- `GET /websites/:websiteId/sessions/:sessionId/activity`：**新增**
- `GET /websites/:websiteId/sessions/:sessionId/properties`：**新增**
- `GET /websites/:websiteId/session-data/properties`：**新增**
- `GET /websites/:websiteId/session-data/values`：**新增**

## 2024-05-23

**网站**

- `GET /websites/[id]/stats`：将 `change` 重命名为 `prev`。

## 2024-03-27

**网站**

- `GET /websites/[id]/stats`：将 `uniques` 重命名为 `visitors`。添加了 `visits`。

## 2024-02-16

**网站**

- `POST /websites`：将 teamId 添加为可选参数。

**团队**

- `/teams/[id]/websites/[websiteId]`：**弃用**
- `GET /teams/[id]/users/[userId]/index`：**新增**

**报告**

- `GET /reports/index`：添加 `websiteId` 和 `teamId` 作为查询参数。
