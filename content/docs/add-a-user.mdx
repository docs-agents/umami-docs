title: 添加用户

Umami 允许您创建独立账户，以便为朋友或客户托管数据。

## 角色

团队角色及相关权限。

- **管理员**：拥有所有权限。
- **用户**：拥有除管理控制台外的所有权限。
- **仅查看**：仅可查看。

以管理员身份登录 Umami，点击侧边导航下拉菜单，然后点击**管理**。

只有**管理员**用户才能访问**管理控制台**。

![image](/images/docs/navbar-settings.png)

然后进入**用户**页面，点击**创建用户**按钮。

![image](/images/docs/user-create.png)

填写表单详情，点击**保存**按钮。

![image](/images/docs/user-create-form.png)

新账户将拥有自己的仪表盘和网站，与**管理员**账户分开。
