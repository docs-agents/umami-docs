---
title: 标签
---

使用 Umami 标签将事件分组在单个网站下，收集数据并启用各种功能。

## 使用场景

- A/B 测试：测试网页或营销活动的不同版本，以查看哪个表现更好。
- 将事件分组以允许在单个网站概览下进行筛选和洞察。

## 使用方法

将 **data-tag** 属性添加到追踪脚本。这将在 payload 中包含 **tag** 列。

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-tag="homepage-layout-a"
></script>
```

您可以为网站的不同部分重用追踪脚本，引入额外的标签。

```html
<script
  defer
  src="http://mywebsite.com/umami.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
  data-tag="homepage-layout-b"
></script>
```
