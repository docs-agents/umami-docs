---
title: 独立 ID
---

Distinct ID 是分配给用户的唯一标识符，可以是匿名的，也可以在用户登录时分配。
这允许您将操作（页面浏览、点击、转化）与跨多个会话的单个用户关联。
如需更广泛的会话识别方式，请查看[会话数据](/docs/tracker-functions#session-data)。

## 使用场景

- 正确归因用户行为。
- 连接跨多个会话或设备的事件。
- 区分重复用户与新用户。
- 根据用户历史构建用户旅程和档案。

## 使用方法

在 payload 中添加 **id** 属性。此属性有 50 个字符的限制。

```json
{
  "payload": {
    "hostname": "your-hostname",
    "language": "en-US",
    "referrer": "",
    "screen": "1920x1080",
    "title": "dashboard",
    "url": "/",
    "website": "your-website-id",
    "id": "bob@aol.com"
  },
  "type": "event"
}
```
