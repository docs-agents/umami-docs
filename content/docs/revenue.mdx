---
title: 收入
---

Umami 收入帮助您追踪财务业绩、衡量投资回报率，并识别哪些页面或产品产生最多收入。
它提供对客户行为的洞察，帮助优化收入来源并提高转化率。
通过将收入数据与流量和用户行为关联，企业可以做出明智的决策来提高盈利能力并规划未来增长。

该分析功能通过在指定时间段内聚合**收入**和**货币**数据来工作。

## 配置收入

要开始收集收入数据，您需要利用动态数据追踪事件。这可以通过[追踪器函数](https://umami.is/docs/tracker-functions#event-data)
或[数据属性](https://umami.is/docs/track-events#using-data-attributes)完成。追踪的事件必须与两个动态数据属性 `revenue` 和 `currency` 一起传递。
如果无法识别货币代码（ISO 4217），分析将默认为 `USD`。

### 追踪器函数

```js
umami.track('checkout-cart', { revenue: 19.99, currency: 'USD' });
```

追踪事件时，默认属性包含在 payload 中。
这相当于运行：

```js
umami.track(props => ({
  ...props,
  name: 'checkout-cart',
  data: {
    revenue: 19.99,
    currency: 'USD',
  },
}));
```

### 数据属性

请注意，使用此方法时收入必须作为字符串传递。

```html
<button
  id="signup-button"
  data-umami-event="checkout-cart"
  data-umami-event-revenue="19.99"
  data-umami-event-currency="USD"
>
  Checkout
</button>
```

## 参数

- `货币`：要聚合的数据的货币（必填）。
