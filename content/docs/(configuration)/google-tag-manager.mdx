---
title: 使用 Google Tag Manager
---

可以使用 Google Tag Manager 安装 Umami 追踪器。

通常，您可以使用以下代码安装 Umami 追踪器：

```html
<script
  defer
  src="http://mywebsite.com/script.js"
  data-website-id="94db1cb1-74f4-4a40-ad6c-962362670409"
></script>
```

但是，Google Tag Manager 会从追踪器中剥离属性。因此您需要改用以下代码：

```javascript
<script>
  (function () {
  var el = document.createElement('script');
  el.setAttribute('src', 'http://mywebsite.com/script.js');
  el.setAttribute('data-website-id', '94db1cb1-74f4-4a40-ad6c-962362670409');
  document.body.appendChild(el);
})();
</script>
```
