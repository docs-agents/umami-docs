---
title: 追踪出站链接
---

当用户点击指向外部网站的链接时，此事件通常不会被捕获，因为
用户正在离开运行 Umami 的网站。但是，使用[事件](/docs/track-events)您可以
追踪此行为。

要追踪出站链接，您需要向包含外部链接的锚点标签添加数据属性。
当点击标签时，事件将被触发。在此示例中，我们发送一个名为 `outbound-link-click` 的事件，
其中 `url` 值设置为外部 URL。

```html
<a href="https://www.external-website.com"
  data-umami-event="outbound-link-click"
  data-umami-event-url="https://www.external-website.com"
>
  External link
</a>
```

如果您不想手动更新所有锚点标签，可以使用此脚本自动将
事件属性添加到所有出站标签。您可以将以下脚本放在 HTML body 的底部。

```html
<script type="text/javascript">
  (() => {
    const name = 'outbound-link-click';
    document.querySelectorAll('a').forEach(a => {
      if (a.host !== window.location.host && !a.getAttribute('data-umami-event')) {
        a.setAttribute('data-umami-event', name);
        a.setAttribute('data-umami-event-url', a.href);
      }
    });
  })();
</script>
```
