Umami 是一个尊重用户隐私的开源网站分析平台。无需 Cookie，不跨站追踪，不收集个人数据。开箱即符合 GDPR。

本文档涵盖 **Umami v3**。如需查看上一版本，请参阅 [v2 文档](https://v2.umami.is)。

## 为什么选择 Umami？

- **隐私优先** — 无 Cookie、无指纹识别、无个人数据。您的访问者保持匿名。
- **拥有您的数据** — 在您的基础设施上自托管。您的分析数据永远不会离开您的服务器。
- **轻量级** — 追踪脚本小于 2KB。不会减慢您的网站速度。
- **简单** — 使用 Docker 在几分钟内部署。无需复杂配置。

## 开始使用

```bash
docker compose up -d
```

就这样。打开 `http://localhost:3000` 并使用 `admin` / `umami` 登录。

有关详细的设置选项，请参阅[安装](/docs/install)。有关托管平台，请查看我们的[部署指南](/docs/guides/hosting)。

## 功能

**核心分析** — 页面浏览量、访问者、跳出率、会话时长、引荐来源、浏览器、操作系统、设备和国家/地区。

**自定义事件** — 使用简单的数据属性或 JavaScript 调用追踪按钮点击、表单提交或任何用户交互。

**高级洞察** — 漏斗、用户旅程、留存分析、目标、UTM 营销活动追踪和队列细分。

**会话** — 查看个人访问者活动和会话属性，无需识别个人信息。

**团队** — 与您的团队协作。分享网站、分配角色和管理访问权限。

**API** — 完整的 REST API，用于以编程方式访问您的所有分析数据。
